<div class="article-description" style="text-align: center; font-size: 1.3em; color: #555; margin: 20px 0 30px 0; font-weight: 500;">又给自己挖了个坑</div>

我是一个很乐于记录的人，用文字记录我的各种想法算是我原始的欲望中的一部份。

在读本科之前，我的日常生活和社交都还算比较有趣，因此能够自娱自乐的记录很多东西，在现在的自己读来也很有意思。但是本科期间，我和他人互动的次数变少了很多，大多数时候还是自己一个人坐在某个位置，面对着电脑屏幕和上面倒映出的自己了。但是记录的欲望一直存在，虽然我更费劲地去尝试捕捉某一些片段，但很显然它们正在以一种不可逆的方式慢慢消失。我在24年开头注册过一个微信公众号，记录过几篇文章，但是大多数是某种负面情绪或者感想的映射。这并不是我刻意而为之，但事实就是如此。

<!-- more -->

## 转变的契机

既然原先记录的话题大多数都已经转向负面，不如将记录的对象重点转移到技术上面。在读完几位友人运营的技术性博客之后，我早早就有了创立一个个人博客的想法，但碍于感觉很耗时间迟迟没有动手。

这一切转变的契机是我开始使用 `Claude Code`，并发现他的能力似乎已经能够让我不怎么动脑的搭建起来一个个人博客了。说干就干，晚上打开的 CC，我选了个模版后大概断断续续 vibe 了三四个小时，终于搭建的像个样子了（btw，欣喜的是这种高强度的 vibe 居然还没有用到我最便宜国产 api 的5小时限额）。

这个模版好在后续的更新都只用写 **Markdown**，然后网页会自动生成，不过我暂时还没有摸索独立于 Github 仓库的独立域名那一块，anyway 先跑着看看吧。

## 博客会有什么

我还不确定这个博客具体会放些什么东西，也许是杂七杂八的技术探索，或者是抽象的思维碎片？

说实话这里给我的感觉更加 **自由**，也许也会放一些娱乐的东西，甚至想写一点乐评或者日常生活分享。

Stay tuned！

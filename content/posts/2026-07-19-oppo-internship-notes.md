七月一号入职，地点在万象前海上面那栋写字楼。离我家很近，骑小摩托几分钟就能到，这算是整段实习里最超模的一点（通勤五分钟，上班八小时！）。O 算是个硬件大厂，但对一个做软件的人来说，可能就不算特别中心。我被分到软工部，系统集成开发里的相机组，主要解决相机五花八门的问题。

前几天主要还是以学习熟悉内部工作流程为主。O 内部的文档系统做得非常好，好到我有时候会怀疑：做文档这件事本身会不会反而成为工作效率的瓶颈。但事实是，做出优秀的文档确实挂着一个 minor 的 KPI。入职前两周主要就是读前人留下的各种教学 —— 实际上根本读不完，至少实习期内不可能，实在是太详尽了。再加上之前的确对 Camera2 API 之类的没啥了解，基本是看到哪算哪的水平。

## 带薪自习

O 对实习生的产出要求并不高。刚开始我都是边熟悉流程边干点杂活，比如刷机、解 BL 锁、跑测试一类。同时 O 的考勤很松，手机定位灵活打卡，早上欠的时间下午补回来就行。于是我就主要处于一种 chill 的带薪自习状态。

最近给我分了一些「解决 UI 效果不符合预期」的任务，正在着手修。干活干着干着感觉我本质上是 Cursor 与效果之间的 bridge。说到 Cursor，组里给了一个 team 账号，额度不太透明，但似乎很难蹬到上限，甚至还能选 fable！组里整体非常忙，每天处理的问题很多，最近赶上新大版本上线期，倒也合理。

## 学历厂

有一天 O 用大巴把我们这批实习生拉去滨海湾的基地，上了一天专属于实习生的企业文化课。滨海湾那个基地修得非常好，是很现代化的公园式工作环境。

也是在那天我发现，O 竟然是个学历厂。院校下限是中九，哈工扎堆，华五的也不在少数，整体以硕士为主。似乎只有我，顶着一个双非皇帝本科生的名号在招摇过市。不过，神奇的是学长面对面环节的学长居然也是南科大的，是材料系的24硕士，可能 O 就是偏爱南科大一点吧。

看 HC 和培养的配置，O 像是相当期望我们转正的，各方面都给人一种认真的感觉。

## 同期的人

认识了几个实习生伙伴，感觉都还不错，交流起来很顺利，不知道是不是面试筛选做到位了。

最神奇的一个在做 kernel 优化相关：本科浙大海洋工程，工作了两年，跨考南大软件，结果因为有工作经历反而被字节 HR 秒挂。还有之前在笛子工作过、现在读港城水硕顺手来实习的。

## 后记

说白了我也没什么转正压力，本就把这当社会化的第一课。唯一意外的是，我所在组干的活，跟学校软工课教的那套流程几乎是相同的，code review 跑冒烟之类的。话说回来，之前的确对软工这门课程评价有点过低了。在前家庭作坊 coding 场景，的确是一个人加上一个 unlimited api 就可以变成自己的超人。然而在真实工业环境，特别是这种偏硬件的公司，代码出问题的成本极高，流程和规范就显得非常重要，也许这就是软工的意义所在吧。

...有点眼馋 agent 岗位了...
